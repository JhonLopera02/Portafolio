#  Portafolio Profesional - Jhon Lopera

Bienvenido a mi portafolio web. Este proyecto muestra mis habilidades en desarrollo frontend mediante la creación de una página web moderna, responsive e interactiva.

---

##  Vista general

Este sitio incluye:

-  Información personal
-  Proyectos destacados
-  Página adicional de mascotas
-  Formulario de contacto
-  Interactividad con JavaScript

---

##  Tecnologías

| Tecnología | Uso |

| HTML5 | Estructura del sitio |
| CSS3 | Estilos y diseño responsive |
| JavaScript | Interactividad |
| Font Awesome | Iconos |

---

##  Responsive Design

El diseño se adapta a distintos dispositivos:

-  **Móviles:** navegación vertical, contenido en una columna  
-  **Tablets:** distribución en dos columnas  
-  **Escritorio:** layout en tres o más columnas  

---

##  Funcionalidades

-  Botón que cambia el texto dinámicamente  
-  Mensaje de bienvenida al cargar la página  
-  Mostrar y ocultar proyectos  
-  Menú responsive interactivo  

---

---

##  Buenas prácticas aplicadas

- Uso de `:root` para variables globales  
- Código limpio y organizado  
- Uso de `rem` para tipografía  
- Imágenes responsivas (`max-width: 100%`)  
- Separación de responsabilidades (HTML, CSS, JS)
